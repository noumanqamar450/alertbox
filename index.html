Documentation

How to install.

Three steps — done in under a minute.

01

Add this <script> tag to your HTML footer 

<script src="https://cdn.jsdelivr.net/gh/noumanqamar450/alertbox@main/version/1.0.2/alertbox.min.js"></script>
02

Basic AlertBox function call 

alertbox.render({
  alertIcon: 'question',
  title: 'Thank You!',
  message: 'AlertBox Popup Message',
  btnTitle: 'Ok',
  themeColor: '#000000',
  btnColor: '#7CFC00',
  border: true
});
03

Custom theme colour 

alertbox.render({
  alertIcon: 'success',
  title: 'Thank You!',
  message: 'AlertBox Popup Message',
  btnTitle: 'Ok',
  themeColor: '#000000'
});

Custom button colour 

alertbox.render({
  alertIcon: 'success',
  title: 'Thank You!',
  message: 'AlertBox Popup Message',
  btnTitle: 'Ok',
  themeColor: '#000000',
  btnColor: '#7CFC00'
});

Enable border (default is false) 

alertbox.render({
  alertIcon: 'success',
